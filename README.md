# Clase 1 Lab

Repositorio base del programa **Desarrollador Senior de Vibe Coding con OpenCode**.

Este proyecto documenta la preparación del entorno profesional de desarrollo.

## Objetivo de la Clase 1

Preparar una computadora Windows con un entorno profesional basado en:

- WSL
- Ubuntu
- Visual Studio Code
- Git
- GitHub
- Docker
- Python
- pip
- venv
- Node.js
- npm
- nvm
- OpenCode

## Objetivo final del programa

Formar desarrolladores senior de vibe coding con OpenCode, capaces de construir 100 proyectos profesionales de agentes IA para problemas reales de negocio.
